Assumptions & Methodology
  • Data source: Jira REST API v3 at redhat.atlassian.net, queried 2026-05-17 via /rest/api/3/search/jql (cursor-based pagination). Only AIPCC project accessible without authentication.
  • INFERENG, RHAIENG, RHOAIENG require Jira authentication. JQL queries return empty result sets without credentials. Configure credentials via Workspace Settings > Integrations to include their data.
  • Release burndown risk: Green = ≥70% done or >30 days remaining, Yellow = <70% done and <30 days remaining, Red = <50% done and <14 days remaining, or overdue.
  • Release issues queried via "Target Version" (cf[10855]), NOT fixVersion. Epic and Feature issue types excluded.
  • Aging WIP: Issues with statusCategory = "In Progress" and updated date >14 days ago. 230 of 628 total WIP items (37%) are stale — slight improvement from 246/631 (39%) on 2026-05-15.
  • WIP:Throughput ratio = current WIP count / 30-day resolved count. Values <1.0 generally healthy; >2.0 indicates flow congestion.
  • Net flow: 30-day created (1,644) minus 30-day resolved (1,500) = +144 net growth. Backlog growth rate worsened slightly from +132 on prior refresh.
  • Known release dates: RHAII-3.5 EA1 = 2026-06-02, rhoai-3.5.EA1 = 2026-06-02, RHAII-3.5 = 2026-06-18, rhoai-3.5.EA2/RHAII-3.5 EA2 = 2026-06-30, rhoai-3.5 = 2026-07-14.
  • rhelai-3.5 EA1/EA2 versions previously tracked now show 0 issues via Target Version. Issues may have been reassigned to different versions since May 15.
  • statusCategory grouping: "Done" = resolved/closed, "In Progress" = active work, "To Do" = not started.
  • The Agile Board API requires authentication and was not used. All data derived from JQL search queries.
