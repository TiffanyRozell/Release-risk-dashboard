Assumptions & Methodology
  • Data source: Jira REST API v3 at redhat.atlassian.net, queried 2026-05-25 via /rest/api/3/search/jql (cursor-based pagination, maxResults=5000 per page).
  • AIPCC is publicly accessible. INFERENG, RHAIENG, RHOAIENG require authentication (JQL returns empty result sets without credentials).
  • Sprint data is live from today's JQL queries using sprint in openSprints(). All 5 teams on Sprint 33 (May 19 – Jun 2). Team assignment derived from Jira component field. 307 total sprint issues across all components. KPIs (WIP, throughput, aging, created) are also live.
  • Sprint risk compares current sprint item count against 3-sprint avg velocity (resolved per 2-week window by component): Green = within +10%, Yellow = 10-30% over, Red = >30% over.
  • Release issues use the "Target Version" custom field (cf[10855]), NOT fixVersion. Issue types Epic and Feature are excluded.
  • Component mapping: Development Platform, AI Testing + Workflow Validation, Accelerator Enablement, AIPCC Productization, PyTorch. Minor components (AIPCC Ecosystems 5 items, Wheel Package Index 1 item, No Component 10 items) not shown as separate cards.
  • Velocity = count of issues resolved per 2-week window by component. DP avg 108, WV avg 53, Accelerators avg 83, Productization avg 67, PyTorch avg 170.
  • Release risk is based on completion percentage: Green = ≥80%, Yellow = 50-79%, Red = <50%.
  • statusCategory grouping: "Done" = resolved/closed, "In Progress" = active work, "To Do" = not started.
  • Changes from May 24: WIP 609→610 (+1, stable), throughput 1,303→1,479 (+176, rolling window shift captures more recent resolutions), created 1,407→1,611 (+204, rolling window shift), aging WIP 249→249 (unchanged). Velocity recalculated with fresh data: DP avg 123→108, WV 55→53, AE 126→83, Prod 84→67, PT 176→170 (slight declines as rolling windows shift). All sprint teams Green (all item counts well within velocity capacity). Release issue counts unchanged from May 24.
  • 14 unreleased versions in AIPCC currently have issues with Target Version set.
