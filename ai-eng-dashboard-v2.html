Assumptions & Methodology
  • Data source: Jira REST API v3 (POST /rest/api/3/search/jql, anonymous access) queried live on 2026-04-20.
  • INFERENG, RHAIENG, RHOAIENG returned 0 issues via anonymous search; data shown only for AIPCC. These projects require authenticated Jira access.
  • Velocity = resolved issues per 2-week window (proxy for sprint velocity since Agile board API requires auth). Computed from 6 trailing 14-day periods.
  • "Target Version" (customfield_10855) used for release mapping, NOT fixVersion.
  • Issue types excluded: Epic, Feature (per release mapping query).
  • Risk thresholds (Release Risk):
    • Green: remaining ≤ avg_velocity × 1.1 (within +10% of avg velocity)
    • Yellow: remaining ≤ avg_velocity × 1.3 (10-30% over avg velocity)
    • Red: remaining > avg_velocity × 1.3 (>30% over avg velocity)
  • Aging WIP = in-progress issues not updated for >14 days. Currently 207 of 547 WIP items (37.8%) are stale.
  • KPI deltas compare to prior refresh (2026-04-19). WIP rose from 399 → 547 (+148). Throughput 1,059 → 1,250. Created 1,199 → 1,404. Avg velocity 494 → 548.
